# Шифратор Виженера

Простое консольное приложение для шифрования и дешифрования текста с помощью шифра Виженера.

## Возможности
- Шифрование и дешифрование текста
- Поддержка русского и английского алфавита
- Загрузка текста из файла
- Сохранение результата в файл

## Как запустить
1. Установите Python 3
2. Скачайте файл `vigenere.py`
3. Запустите:
   ```bash
   python vigenere.py
